Introducción a React js

¿Que es React?

React es un framework de desarrollo web. Utiliza javascript y css, pero la interfaz de usuario se escribe con un lenguaje de maquetación distinto a HTML, utiliza JSX. Su sintáxis es similar a HTML pero está basada en componentes y es sencillo de crearlos y agruparlos, además que permite instertar javascript en el mismo archivo de una manera elegante.

Cómo funciona

  1. index.html ... body: div id:'root' :body ...
  2. index.js
    • import React from 'react'
    • import ReactDOM from 'reactDOM'
    • import App from './App'
    • ReactDOM.render(App, document.getElementById('root')

Así se conecta la aplicación React con el html, a través del index.js que renderiza el componente App que, generalmente, contiene a todos los otros componentes del sistema
